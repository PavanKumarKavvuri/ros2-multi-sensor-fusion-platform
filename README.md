# Multi-Sensor Robot Integration (ROS2)

A modular robotics platform integrating IMU, LiDAR, and camera sensors using ROS2, focused on real-world challenges in time synchronisation, pose estimation, and multi-sensor fusion.

## Key Focus Areas

* Sensor integration (IMU, LiDAR, camera)
* Time synchronisation and data alignment
* Pose estimation from individual modalities
* Foundations for sensor fusion

## Why this project?

Most robotics projects demonstrate algorithms in isolation. This project focuses on making multiple sensors work together reliably in a real system, addressing timing, drift, and data consistency challenges.

## Current Status

* IMU + LiDAR + Camera integrated via ROS2
* Sensor synchronisation experiments in progress
* Pose estimation pipelines planned
* Sensor fusion (EKF / graph-based) planned
